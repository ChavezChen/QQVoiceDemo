# QQVoiceDemo
高仿QQ录音功能模块

本篇代码实现的功能：
- 1、封装AVAudioRecorder以及AVAudioPlayer实现录音以及播放功能。
- 2、实现录音以及播放时的振幅动画。
- 3、播放时环形进度条的动画。
- 4、变声效果。
